Potato Cholesterol App

To Run:

1. Open FIT3077Potato in Browser
2. Use ID Number test case "4183988" and click "Get Patient List"
3. Tick selection to view cholesterol values
4. In Cholesterol Threshold, use test cases "180" and "200" to view values above and below threshold

Additional note:
The default threshold value prevents cholesterol values above a certain value to be uinhighlighted, no matter
the cholesterol threshold input
